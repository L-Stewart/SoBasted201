# H3 Description of the project: 
# H4 A website that allows a user to build a menu and creates a shopping list
# H3 Problem domain and how the project resolved those problems: 
- Create a selection of recipes on the "Recipes" page that a user may choose to use 
- Create a table of the chosen recipes, where the user can choose to revise by removing chosen recipes while on the "Menu" page, or go back to the "Recipes" page to add more recipes
- Create a table of the total amount per ingredients for the "Shopping List" page
- Finally, creates an option for the user to clear all their selected recipes and go back to the "Recipes" page
